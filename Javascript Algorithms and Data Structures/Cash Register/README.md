# Cash Register

This project is a solution to the [Build a Cash Register](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures-v8/build-a-cash-register-project/build-a-cash-register) challenge from freeCodeCamp.

## Overview

The Cash Register app allows users to input purchase prices and cash provided, then calculates and displays the correct change using available currency denominations.

## Features

-   User-friendly interface built with HTML and CSS
-   JavaScript logic to calculate change and handle edge cases
-   Responsive design for various screen sizes

## Usage

1. Enter the purchase price and cash provided.
2. Click the button to calculate change.
3. The app displays the change breakdown or relevant messages.

## Project Files

-   `index.html` – Main HTML structure
-   `styles.css` – Styling for the app
-   `script.js` – Cash register logic
